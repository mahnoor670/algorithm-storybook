# Algorithm Storybook

A full-stack educational web application that teaches sorting 
and searching algorithms to high school students and 
introductory CS learners through interactive, story-driven 
game levels.

## Tech Stack
- **Frontend:** React, JavaScript, HTML/CSS
- **Backend:** Python, Flask
- **Architecture:** REST API

## Algorithms
| Algorithm | Level | Status |
|-----------|-------|--------|
| Binary Search | Acorn Hunt | Live |
| Bubble Sort | Class Photo | Live |
| Selection Sort | Professor's Stack | Live |
| Insertion Sort | TBD | Coming Soon |
| Merge Sort | TBD | Coming Soon |
| Quick Sort | TBD | Coming Soon |

## How to Run

### Backend
```bash
pip install flask
python app.py
```

### Frontend
```bash
cd client
npm install
npm start
```

App runs at http://localhost:3001
